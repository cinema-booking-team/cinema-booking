# cinema-booking
короче, чтобы все нормально запустилось.

запустите PyCharm, и установите все ниже:
django==4.2
djangorestframework
django-cors-headers
mysqlclient

я использовал питон версии 3.12 (у меня траблы были, пришлось устанавливать более старую версию, не знаю, запустится ли на новой)

также поднимите бд(файл с выгрузкой у нас в тг), я через xampp делал, порт MySQL: 3306. В СУБД создайте БД с названием cinemabook и импортируйте выгрузку
после импорта выгрузки пропишите в терминал это: python manage.py migrate --fake

далее открываете 2 терминала и прописываете в каждом отдельно:
python manage.py runserver - на запуск апи
python -m http.server 3000 - на запуск фронта

и в браузере открываете http://127.0.0.1:3000/pages/movies.html
